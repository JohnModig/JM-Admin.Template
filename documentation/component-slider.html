Slider Optional

Scroll through elements horizontally, with navigation and progress indicator. This component is usually used to create carousel slideshows or to divide long forms into smaller sections.

Features
  • Automatically creates a navigation with optional styles; "bullets" or "previous and next buttons".
  • Automatically creates a progress indicator with optional description of the steps.
  • Can be scrolled automatically.
  • Can be translated into other languages.
Configuration
  • Sass file: components/_slider.scss
  • Javascript file: components/_slider.js
  • Include the compiled css file in the Html: <link href="/css/components/slider.min.css" rel="stylesheet" />
  • Include the compiled javascript file in the Html: <script src="/javascript/slider.min.js"></script>
Implementation Javascript
Basic

Create a Slider object with default configuration:

const mySlider = new Slider('#mySlider');

Advanced

Create a Slider object with detailed configuration:

const mySlider = new Slider({ element: '#mySlider', gap: '100px', navigationStyle: 'prevNext', steps: ['Introduktion', 'Detaljer', 'Summering'], appendElement: 'a.success', text: { Slide: 'Bild', Previous: 'Föregående', Next: 'Nästa' } });
