Loader

Animated graphics to display while loading content. This component is usually used by javascript to indicate loading in progress.

Features
  • CSS only.
  • Can be inserted into any Html element.
  • Implement by setting the class="loading" attribute of the Html tag.
  • Used by the dialog component.
