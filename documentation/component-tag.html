Tag Optional

A small label or link. This component is usually used to flag or indicate a status to a subject.

Features
  • Can be text (<span>) or a link (<a>).
  • Can have an icon, via the themes icon font (class="icon-…).
  • Can be in highlighted color (class="highlighted").
  • Supports alert colors.
Configuration
  • Sass file: components/_tag.scss
  • Include the Sass file in theme.scss: @import "components/tag";

Variants: Text only, icon and text, icon only.
