Icons

Styles for the custom made icon web font. The set includes icons for basic functionality of the website (e.g. menu, close, add, delete) as well as some icons for basic information (e.g. info, error, warning).

This font includes a limited set of icons for performance reasons. If more icons are needed, we recommend to put these in a separate web font.

Features
  • Web font.
  • CSS only.
  • Implement by setting the class attribute of the Html tag.
  • Works with typographic elements (e.g. <h1>, <h2>, <h3>, <h4>, <h5>, <p>, <li>, <a>, <span>, <summary>, <sup>).
  • Works with other components (e.g. Badge and Horizontal tabs).
  • Influenced by (but not limited to) Material Icons, outlined.
